22.04. Введение в теорию нечётких множеств. Лотори Заде

(5++ за лекцию)

1. Основные понятия
Теория нечётких мн-в — раздел математики, изучающий различные размытые понятия, которые использует человек, принимая решения в реальных условиях.

Нечёткое множество (fuzzy set) — совокупность пар вида:
А = { (x, мюA(x))  |  x  принадлежит  U }
где:
 U — универсальное мн-во (универсум)
 мюA(x) — ф-ия принадлежности, которая ставит в соответствие кажд. элем. x  принадлежит  U число из [0; 1].

Характеристики нечётких мн-в:
 Носитель (суппорт) — чёткое подмн. U, элем. которого имеют ненулевые степени принадлежности.
 Высота — верхняя граница его ф-ии принадлежности (если выс = 1, то мн-во называется нормальным, если <1 — субнормальным).
 Ядро нечётк. мн. — чёткое подмн-во U, элем. которого имеют степень принадлежности = 1.
 Точки перехода — элементы U, для которых мюA(x) = 0.5.
 альфа-сечение (альфа-срез) — чёткое подмн-во U, элем. которого имеют степень принадлежности в интервале альфа  принадлежит  [0, 1].

2. Функция принадлежности в теор. нечёт. мн-в
Это обобщение индикаторной (характеристической) ф-ии классич. мн-ва в рамках теор. нечётких мн-в. Она показывает степень принадлежности каждого элемента пространства рассуждений (темы) к данному нечётному мн-ву.

Основные характеристики:
 Функция принадлежности мюA(x) принимает значения из [0; 1]:
   если мюA(x) = 0 — то элем x  не принадлежит  нечётному мн-ву;
   если мюA(x) = 1 — то элем x полностью  принадлежит  нечётк мн-ву;
   если 0 < мюA(x) < 1 — то элем x  принадлежит  мн-ву частично.
 Нормальность: нечёткое мн-во называется нормальным, если существует хотя бы один элемент x  принадлежит  X, для которого мюA(x) = 1.

Примечания: (подмножество A  подмножество  X)

 Индикаторная (характеристическая) ф-ия — функция 1_A: X  ->  {0, 1} (я хз что это значит), которая указывает на принадлежность x  принадлежит  X подмножеству A.
 В классической теории множеств элемент либо принадлежит множеству ( принадлежит ), либо нет ( не принадлежит ). В теории нечётких множеств функция принадлежности расширяет это понятие и позволяет градуировать принадлежность, что особенно важно при описании расплывчатых понятий.

29.04. Методы построения функции принадлежности

Методы делятся на две группы: прямые и косвенные.

3. Методы построения ф-ии принадлежности

 Прямые методы характеризуются тем, что эксперт непосредственно задаёт правила определения значений функции принадлежности, характеризующей элемент x. Эти значения согласуются с его предпочтениями на множестве элементов X следующим образом:
   Для любых x1, x2  принадлежит  X: мюA(x1) < мюA(x2) тогда и только тогда, когда x2 предпочтительнее x1 (т.е. в большей степени характеризуется свойством А).
   Для любых x1, x2  принадлежит  X: мюA(x1) = мюA(x2) тогда и только тогда, когда x1 и x2 безразличны относительно свойства А.
   Прямые методы, как правило, используются для таких понятий (физических параметров) как: v, t, S, t° (физические обозначения) и т.п.

 Косвенные методы используются в тех случаях, когда отсутствуют элементарные измеримые свойства нечёткого множества.
   К косвенным методам относятся: статистический, метод ранжирования с нормировкой, методы оптимизации, теория игр, шкала Саати.
   Примечание: в некоторых случаях для построения функции принадлежности используется аппроксимация, когда дискретные данные аппроксимируются типовой функцией.

4. Типовые функции принадлежности
Типизация функции принадлежности существенно упрощает аналитические и численные рассчёты при задании ф-ии принадлежности.

Существуют следующие типовые функции принадлежности:
 Треугольная: используется для понятий типа «примерно равно» или «среднее значение».
 Трапециидальная: используется для диапазонов.
 S-образная: описывает понятия «большое значение» или «высокий уровень».
 Z-образная: «малые значения» или «низкий уровень».
 Гауссова: моделирует понятие «приблизительно равно» с плавным убыванием.

Примеры функций принадлежности:

Треугольная («почти 10»):

  μA(x)
   ▲      "почти 10"
  1 ┼       /\
    │      /  \
 0.6┼ - - / - -\- -
    │    /      \
 0.3┼ - / - - -  \
    │  /          \
  0 └──┴────┴──┴──┴──► x
       0    5  10 11

Гауссова:

  μA(x)
   ▲
  1 ┼       .---.
    │      /: ║ :\
    │     / : ║ : \   (пунктиром показана более узкая
    │    /  : ║ :  \   колоколообразная кривая)
    │   /   : ║ :  \
  0 └──┴────┴─╨─┴────┴► x

5. Лингвистическая переменная
 Функция принадлежности — невероятностная субъективная мера нечёткости, определяемая по результату опроса экспертов. В отличие от вероятностной меры, имеющей дело с неопределённостью наступления некоторого события, нечёткая мера выступает численной оценкой лингвистической неопределённости, связанной с неоднозначностью и расплывчатостью человеческого мышления.
 Лингвистическая переменная — переменная, значениями которой являются не числа, а слова или фразы естественного или искусственного языка.

Задаётся кортежем:
 <  x, T, U, G, M  > 
где:
 x — имя переменной;
 T — терм-множества;
 U — универсум;
 G — синтаксическое правило для образования новых значений переменной;
 M — семантическая процедура, которая связывает новое имя переменной с нечётким мн-ом.

 Терм-множество — мн-во всех возможных значений лингвистической переменной.
 Терм — любой элем. терм-множества.

Пример: "Теплая погода"
 Если t°C = 25^\circ, то мю{тёплая}(25) = 1 "полностью теплая"
 Если t°C = 18^\circ, то мю{тёплая}(18) = 0,7 "скорее теплая"
 Если t°C = 10^\circ, то мю_{тёплая}(10) = 0,2 "почти не теплая"
Тогда: А = {(25; 1); (18; 0,7); (10; 0,2)}

06.05. Основы нечёткой логики

1. Основы нечёткой логики
Нечёткая логика (fuzzy logic) — логическое исчисление, расширяющее классическую бинарную логику и позволяющее работать с неопределённостью и размытыми понятиями. 

В отличие от классической логики, где высказывание может быть только истинным (1) или ложным (0), в нечёткой логике степень истинности принимает любое значение из интервала [0; 1].

Нечёткое высказывание — повествовательное предложение, относительно которого можно судить об истинности или ложности только с некоторой степенью уверенности.

Степень истинности выражается числом из интервала [0; 1]:
 0 — абсолютная ложность;
 1 — абсолютная истинность;
 между 0 и 1 — частичная истинность.

Пример: дано высказывание «Температура в комнате довольно высокая». Это высказывание может иметь степень истинности, например, 0.7.

 Нечёткий предикат — это функция, зависящая от одной или нескольких переменных и принимающая значение степени истинности из интервала [0; 1].

2. Нечёткие высказывания и логические операции над ними

Классическая логика (чёткая):
Высказывания, которые получаются из элементарных с помощью грамматических связок («не», «и», «или», «если... то...», «тогда и только тогда»), называют сложными или составными.

В алгебре логики для построения сложных высказываний из элементарных используются следующие логические операции: отрицание ( не  или черта сверху), конъюнкция ( и ), дизъюнкция ( или ), импликация ( -> ), эквиваленция ( <-> ).

\ пояснения определений всех операций \

Нечёткая логика (нечёткие высказывания):
Нечётким высказыванием A называется любое утверждение, которое истинно или ложно в определённой степени.

Функция истинности лямбда(А)* — функция, которая ставится в соответствие каждому нечётному высказыванию и принимает значения из интервала [0; 1].

Пусть A и B — нечёткие высказывания, а лямбда(А) и лямбда(В) — соответствующие им степени истинности. Тогда нечёткие логические операции на множестве нечётких высказываний определяются следующим образом:
 Отрицанием нечёткого высказывания A называется нечёткое высказывание не A, степень истинности которого равна:
  лямбда(не А) = 1 - лямбда(А)
 Конъюнкцией A и B называется нечёткое высказывание A  и  B, степень истинности которого совпадает со степенью истинности наименее истиного высказывания:
  лямбда(А и В) = min(лямбда(А); лямбда(В))
 Дизъюнкцией A и B называется нечёткое высказывание A  или  B, степень истинности которого совпадает со степенью истинности наиболее истиного высказывания:
  лямбда(А и В) = max(лямбда(А); лямбда(В))

13.05. Принятие решений на основе нечёткой логики

(5++ за приветствие)

 Импликацией A и B называется нечёткое высказывание A  ->  B, степень истинности которого определяется выражением:
  лямбда(А -> В) = max{1 - лямбда(А); лямбда(В)}
  Степень истинности импликации не меньше, чем степень ложности её посылки или степень истинности её следствия.
 Эквиваленцией A и B называется нечётк. высказ. A  ->  B, степень истинности которого определяется выражением:
  лямбда(А -> В) = min{max{1 - лямбда(А); лямбда(В)}, max{1 - лямбда(В); лямбда(А)}}
  Степень истинности эквиваленции совпадает со степенью истинности менее истинной из импликаций A  ->  B и B  ->  A.

Из нечётких высказываний при помощи логических операций можно строить составные нечёткие высказывания, степень истинности которых рассчитывается по приведённым выше формулам.

Порядок выполнения операций над нечёткими высказываниями: скобки, не {},  и ,  или ,  -> ,  <-> .

3. Этапы нечёткого вывода
Процесс нечёткого вывода включает следующие этапы:
1. Фазификация — преобразование чётких входных данных в нечёткие множества с помощью функций принадлежности.
2. Агрегирование условий — вычисление степени истинности с использованием нечётк. логич. операций.
3. Активизация заключения — применение правил к входным данным.
4. Аккумуляция — объединение результатов всех активированных правил в одно нечётк. мн-во.
5. Дефазификация — преобразование нечётк. результата в чёткое значение.

4. Применение нечёткой логики:
 Системы автоматического управления;
 экспертных системах;
 Распознавание образов;
 Принятие решений в условиях неопределённости;
 Моделирование человеческого мышления;
 в бытовых приборах.

Нечёткая логика нашла широкое применение в технике, экономике, медицине и др. областях, где классическая бинарная логика оказывается слишком «грубой» для описания реальных ситуаций.

27.05. Задача принятия решения группой экспертов, характеризуемых весовыми коэф.

Задача 3. Принятие решений группой экспертов
Постановка задачи:
Лицу, принимающему решение ((ЛПР), требуется принять решение по некоторой проблеме на множестве альтернатив:
U = {u1, u2, ..., un}
где ui — возможное решение проблемы (альтернатива).

ЛПР имеет мнение m экспертов e1, e2, ..., em относительно существующих альтернатив в виде нечётких отношений. Каждый эксперт задаёт своё нечёткое отношение Qk в виде матрицы M{Qk}, в которой выражает степень предпочтения одной альтернативы над другой.

ЛПР по-разному относится к мнению экспертов, что находит отражение в весовых коэффициентах лямбда1, лямбда2, ..., лямбдаm, то есть ЛПР каждому эксперту присваивает коэф. лямбдаi (0  <=  лямбдаi  <=  1).

Требуется принять решение по рассматриваемой проблеме с учётом мнения экспертов относительно множества альтернатив и отношения ЛПР к экспертам.

Алгоритм решения задачи:
1. Найти нечёткое отношение нестрогого предпочтения Q как пересечение нечётких отношений Q1, Q2, ..., Qm.
2. Найти нечёткое отношение Q^{-1}, обратное к нечётному отношению Q.
3. Найти разность нечётких отношений Q и Q^{-1}, которая описывает отношение строгого предпочтения Qs и выражается матрицей отношений M{Qs}.
4. В каждом столбце матрицы M{Qs} найти максимальное значение и вычесть его из 1. В результате будет получено нечёткое подмножество недоминируемых альтернатив U{QND}.
5. Найти комбинацию K нечётких отношений Q1, Q2, ..., Qm относительно весовых коэффициентов:
   K = лямбда1  *  Q1 + лямбда2  *  Q2 + ... + лямбдаm  *  Qm
   где K — нечёткое отношение, в котором интегрально отражены мнения всех экспертов с учётом отношения ЛПР к их мнению.
6. Найти нечёткое отношение K^{-1}, обратное к K, и составить матрицу отношений K^{-1} M{K^{-1}}.
7. Найти разность нечётких отношений K и K^{-1}, результатом которой является нечёткое отношение строгого предпочтения Ks и соответствующая матрица M{Ks}.
8. В каждом столбце матрицы M{Ks} найти максимальный элемент и вычесть его из 1. В результате будет получено нечёткое подмножество недоминируемых альтернатив U{KND}, которое выражает степень недоминируемости альтернатив в контексте интегрального отношения всех экспертов к альтернативам с учётом решения ЛПР ко всем экспертам.
9. Найти пересечение нечётких множеств U{QND} и U{KND}, результатом которого является множество U^ недоминируемых альтернатив в контексте отношений Q и K.
10. Принятие решения: среди элементов множества U^ выбирается та альтернатива ui, которая имеет максимальное значение функции принадлежности мю(Ui).

03.06. Метод анализа иерархии (МАИ)

1. Основные понятия
Метод анализа иерархий (МАИ) — математический инструмент системного подхода к сложным проблемам принятия решений.

МАИ не предписывает ЛПР какое-либо «правильное» решение, а позволяет ему в интерактивном режиме найти вариант (альтернативу), который наилучшим образом согласуется с пониманием сути проблемы.

МАИ позволяет понятным и рациональным образом структурировать сложную проблему принятия решений в виде иерархий.

Анализ проблемы принятия решений в МАИ начинается с построения иерархической структуры, которая включает цель, критерии, альтернативы и др. факторы, влияющие на принятие решений. Анализ ситуации выбора решения в МАИ напоминает процедуры и методы аргументации, которые используются на интуитивном уровне.

2. Этапы решения задач принятия решений в МАИ:
1. Построение качественной модели решаемой проблемы в виде иерархии, включающую цель, альтерн. варианты достижения цели и критерии оценки качества альтернатив.
2. Определение приоритетов всех элементов иерархии с использованием метода парных сравнений.
3. Синтез глобальных приоритетов альтернатив путём линейной свёртки приоритетов иерархии.
4. Проверка суждений на согласованность.
5. Принятие решения на основе полученных результатов.

При принятии управленческих решений с помощью МАИ ЛПР обращается к экспертам, которые создают иерархическую структуру решаемой проблемы.

Иерархическая структура — графическое представление проблемы в виде перевёрнутого дерева, где каждый элемент зависит от выше расположенных элементов. Связь «родитель — ребёнок» направлена сверху вниз.

Рис 1. Иерархическая структура МАИ (альтернативы связаны по принципу «все со всеми» с критериями).

Вершиной иерархии является главная цель. Элементы самого нижнего уровня — альтернативы. Промежуточные уровней представляют собой критерии (факторы), которые связывают цель с альтернативой.

На практике не существует установленной процедуры генерирования элементов иерархической структуры. Построение иерархии исходит из естественной способности человека думать логически и творчески, определять события и устанавливать соответствия между ними.

После построения иерархии определяются приоритеты узлов на каждом уровне:
 Приоритеты — числа, представляющие относительные веса элементов каждой группе сравнения на каждом уровне иерархия.
 Приоритеты являются безразмерными величинами от 0 до 1.

Чтобы найти приоритеты узлов иерархии, составляются матрицы сравнений относительной важности. Каждый элемент строки матрицы сравнивается с элементом столбца. При сравнении с самим собой отношение - 1.

Математическая обработка составленных матриц (суждений) заключается в вычислении вектора локальных приоритетов. Этот вектор выражает относительную силу (величину, ценность) каждого объекта, которые сравниваются в матрице.

Локальные приоритеты перемножаются на приоритет соответствующего критерия на вышестоящем уровне и суммируются по каждому элементу. В результате получается вектор глобальных приоритетов, в котором каждая компонента — это глобальный приоритет соответствующего кандидата (альтернативы).

Кандидат соответствующий самой большой компоненте глобального приоритета будет являться предпочтительным в выборе.
