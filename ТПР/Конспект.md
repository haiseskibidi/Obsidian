## Первый рейтинг

**Градиентом** функции *f(x1, x2, ..., xn)* в точке P называется вектор, составленный из частных производных заданной функции, который направлен по нормали в данной точке в сторону возрастания (убывания) заданной функции:

![[Pasted image 20260619132821.png]]

Свойства:
1. Направление: Направление градиента указывает на направление максимального (минимального) возрастания (убывания) функции в данной точке.
2. Скорость: Модуль градиента |delta f| показывает скорость возрастания (убывания) заданной функции
3. Экстремум: В точках экстремума градиент равен нулевому вектору: delta f = 0

**Матрица Гессе** - это симметричная квадратная матрица n x n, состоящая из частных производных второго порядка функции *f(x)*, вычисленных в заданной точке.

![[Pasted image 20260619133127.png]]

Симметричная матрица А (n x n) и соответствующая ей матрица Гессе могут быть:
1. Положительно определённой (A > 0)
2. Отрицательно определённой (А < 0)
3. Положительно полуопределённой (А >= 0)
4. Отрицательно полуопределённой (А <= 0)
5. Неопределённой (А <> 0)

**Классификация матрицы гессе (Теорема Сильвестра)**: Для классификации используются угловые миноры d1, d2, ... , dn.
1. Чтобы матрица А была **положительно определённой**, необходимо и достаточно: d1 > 0, d2 > 0, dn > 0
2. Чтобы матрица А была **отрицательно определенной**, необходимо и достаточно: d1 < 0, d2 > 0, d3 < 0... (знаки чередуются, начиная с отрицательного).
3. Чтобы матрица А была **положительно полуопределенной**, необходимо и достаточно, чтобы все главные миноры матрицы А были неотрицательными.
4. Чтобы матрица А была **отрицательно полуопределенной**, необходимо и достаточно, чтобы все главные миноры А четного порядка были неотрицательными, а нечетного порядка — неположительными.

### Пример нахождения градиента

**Найти градиент функции:** $f(x, y) = x^2 - 3xy + 2y^2$

*Решение:*

1. $\partial f / \partial x = 2x - 3y$
2. $\partial f / \partial y = -3x + 4y$
3. **Ответ:** $\nabla f(x, y) = (2x - 3y, -3x + 4y)$ (Если есть точка, подставляем координаты)
