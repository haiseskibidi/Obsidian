# 29.04. Методы построения функции принадлежности

Методы делятся на две группы: **прямые** и **косвенные**.

### 3. Методы построения ф-ии принадлежности

* **Прямые методы** характеризуются тем, что эксперт непосредственно задаёт правила определения значений функции принадлежности, характеризующей элемент $x$. Эти значения согласуются с его предпочтениями на множестве элементов $X$ следующим образом:
  * Для любых $x_1, x_2 \in X$: $\mu_A(x_1) < \mu_A(x_2)$ тогда и только тогда, когда $x_2$ предпочтительнее $x_1$ (т.е. в большей степени характеризуется свойством $\tilde{A}$).
  * Для любых $x_1, x_2 \in X$: $\mu_A(x_1) = \mu_A(x_2)$ тогда и только тогда, когда $x_1$ и $x_2$ безразличны относительно свойства $\tilde{A}$.
  * *Прямые методы, как правило, используются для таких понятий (физических параметров) как: $v, t, S, t^\circ$ (физические обозначения) и т.п.*

* **Косвенные методы** используются в тех случаях, когда отсутствуют элементарные измеримые свойства нечёткого множества.
  * К косвенным методам относятся: статистический, метод ранжирования с нормировкой, методы оптимизации, теория игр, шкала Саати.
  * *Примечание: в некоторых случаях для построения функции принадлежности используется аппроксимация, когда дискретные данные аппроксимируются типовой функцией.*

### 4. Типовые функции принадлежности
Типизация функции принадлежности существенно упрощает аналитические и численные рассчёты при задании ф-ии принадлежности.

Существуют следующие типовые функции принадлежности:
* **Треугольная**: используется для понятий типа «примерно равно» или «среднее значение».
* **Трапециидальная**: используется для диапазонов.
* **S-образная**: описывает понятия «большое значение» или «высокий уровень».
* **Z-образная**: «малые значения» или «низкий уровень».
* **Гауссова**: моделирует понятие «приблизительно равно» с плавным убыванием.

#### Примеры функций принадлежности:

**Треугольная («почти 10»):**
```text
  μ_A(x)
   ▲      "почти 10"
  1 ┼       /\
    │      /  \
 0.6┼ - - / - -\- -
    │    /      \
 0.3┼ - / - - -  \
    │  /          \
  0 └──┴────┴──┴──┴──► x
       0    5  10 11
```

**Гауссова:**
```text
  μ_A(x)
   ▲
  1 ┼       .---.
    │      /: ║ :\
    │     / : ║ : \   (пунктиром показана более узкая
    │    /  : ║ :  \   колоколообразная кривая)
    │   /   : ║ :  \
  0 └──┴────┴─╨─┴────┴► x
```

### 5. Лингвистическая переменная
* Функция принадлежности — невероятностная субъективная мера нечёткости, определяемая по результату опроса экспертов. В отличие от вероятностной меры, имеющей дело с неопределённостью наступления некоторого события, нечёткая мера выступает численной оценкой лингвистической неопределённости, связанной с неоднозначностью и расплывчатостью человеческого мышления.
* **Лингвистическая переменная** — переменная, значениями которой являются не числа, а слова или фразы естественного или искусственного языка.

Задаётся кортежем:
$$\langle x, T, U, G, M \rangle$$
где:
* $x$ — имя переменной;
* $T$ — терм-множества;
* $U$ — универсум;
* $G$ — синтаксическое правило для образования новых значений переменной;
* $M$ — семантическая процедура, которая связывает новое имя переменной с нечётким мн-ом.

* **Терм-множество** — мн-во всех возможных значений лингвистической переменной.
* **Терм** — любой элем. терм-множества.

Пример: "Теплая погода"
* Если $t^\circ\text{C} = 25^\circ$, то $\mu_{\text{тёплая}}(25) = 1$ "полностью теплая"
* Если $t^\circ\text{C} = 18^\circ$, то $\mu_{\text{тёплая}}(18) = 0,7$ "скорее теплая"
* Если $t^\circ\text{C} = 10^\circ$, то $\mu_{\text{тёплая}}(10) = 0,2$ "почти не теплая"
Тогда: $\tilde{A} = \{(25; 1); (18; 0,7); (10; 0,2)\}$
