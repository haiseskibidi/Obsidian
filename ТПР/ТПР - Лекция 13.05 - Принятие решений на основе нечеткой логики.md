# 13.05. Принятие решений на основе нечёткой логики

*(5++ за приветствие)*

* **Импликацией** $A$ и $B$ называется нечёткое высказывание $A \rightarrow B$, степень истинности которого определяется выражением:
  $$\lambda(A \rightarrow B) = \max(1 - \lambda(A); \lambda(B))$$
  *Степень истинности импликации не меньше, чем степень ложности её посылки или степень истинности её следствия.*
* **Эквиваленцией** $A$ и $B$ называется нечёткое высказывание $A \leftrightarrow B$, степень истинности которого определяется выражением:
  $$\lambda(A \leftrightarrow B) = \min\Big(\max(1 - \lambda(A); \lambda(B)), \max(1 - \lambda(B); \lambda(A))\Big)$$
  *Степень истинности эквиваленции совпадает со степенью истинности менее истинной из импликаций $A \rightarrow B$ и $B \rightarrow A$.*

Из нечётких высказываний при помощи логических операций можно строить составные нечёткие высказывания, степень истинности которых рассчитывается по приведённым выше формулам.

**Порядок выполнения операций над нечётками высказываниями:**
1. Выражения в скобках;
2. Отрицание ($\bar{A}$);
3. Конъюнкция ($\wedge$);
4. Дизъюнкция ($\vee$);
5. Импликация ($\rightarrow$);
6. Эквиваленция ($\leftrightarrow$).

### 3. Этапы нечёткого вывода
Процесс нечёткого вывода включает следующие этапы:
1. **Фазификация** — преобразование чётких входных данных в нечёткие множества с помощью функций принадлежности.
2. **Агрегирование условий** — вычисление степени истинности условий для каждого правила с использованием нечётких логических операций.
3. **Активизация заключения** — применение правил (определение степеней истинности подзаключений) к входным данным.
4. **Аккумуляция** — объединение результатов всех активированных правил в одно результирующее нечёткое множество.
5. **Дефазификация** — преобразование нечёткого результирующего множества в чёткое числовое значение.

### 4. Применение нечёткой логики:
* Системы автоматического управления;
* Экспертные системы;
* Распознавание образов;
* Принятие решений в условиях неопределённости;
* Моделирование человеческого мышления;
* Бытовые приборы.

*Нечёткая логика нашла широкое применение в технике, экономике, медицине и др. областях, где классическая бинарная логика оказывается слишком «грубой» для описания реальных ситуаций.*
