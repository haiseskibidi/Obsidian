# Первый рейтинг

**Градиентом** функции *f(x1, x2, ..., xn)* в точке P называется вектор, составленный из частных производных заданной функции, который направлен по нормали в данной точке в сторону возрастания (убывания) заданной функции:

![[Pasted image 20260619132821.png]]

Свойства:
1. Направление: Направление градиента указывает на направление максимального (минимального) возрастания (убывания) функции в данной точке.
2. Скорость: Модуль градиента |delta f| показывает скорость возрастания (убывания) заданной функции
3. Экстремум: В точках экстремума градиент равен нулевому вектору: delta f = 0

**Матрица Гессе** - это симметричная квадратная матрица n x n, состоящая из частных производных второго порядка функции *f(x)*, вычисленных в заданной точке.

![[Pasted image 20260619133127.png]]

Симметричная матрица А (n x n) и соответствующая ей матрица Гессе могут быть:
1. Положительно определённой (A > 0)
2. Отрицательно определённой (А < 0)
3. Положительно полуопределённой (А >= 0)
4. Отрицательно полуопределённой (А <= 0)
5. Неопределённой (А <> 0)

**Классификация матрицы гессе (Теорема Сильвестра)**: Для классификации используются угловые миноры d1, d2, ... , dn.
1. Чтобы матрица А была **положительно определённой**, необходимо и достаточно: d1 > 0, d2 > 0, dn > 0
2. Чтобы матрица А была **отрицательно определенной**, необходимо и достаточно: d1 < 0, d2 > 0, d3 < 0... (знаки чередуются, начиная с отрицательного).
3. Чтобы матрица А была **положительно полуопределенной**, необходимо и достаточно, чтобы все главные миноры матрицы А были неотрицательными.
4. Чтобы матрица А была **отрицательно полуопределенной**, необходимо и достаточно, чтобы все главные миноры А четного порядка были неотрицательными, а нечетного порядка — неположительными.

### Пример нахождения градиента

**Найти градиент функции:** $f(x, y) = x^2 - 3xy + 2y^2$

*Решение:*

1. $\partial f / \partial x = 2x - 3y$
2. $\partial f / \partial y = -3x + 4y$
3. **Ответ:** $\nabla f(x, y) = (2x - 3y, -3x + 4y)$ (Если есть точка, подставляем координаты)

## Нелинейная оптимизация

**Евклидово пространство** Rⁿ — пространство с нормой $\|x\| = \sqrt{x_1^2 + x_2^2 + \dots + x_n^2}$.

> Простыми словами: Rⁿ — это просто «координатная система» с n осями. Норма $\|x\|$ — длина вектора от начала координат до точки x (обобщение теоремы Пифагора на n измерений). R² — плоскость, R³ — привычное 3D-пространство.

**Задача**: найти $x^* \in X$, при котором $f(x^*) = \min f(x)$ (или max) на множестве допустимых решений.

> $f(x)$ — **целевая функция**, то, что мы хотим минимизировать/максимизировать (прибыль, затраты, расстояние…). $X$ — множество допустимых точек (область, где можно искать). $x^*$ — та самая «лучшая» точка-ответ.

- **Глобальный min** — $f(x^*) \leq f(x)$ для *всех* $x \in X$ — самая низкая точка на всём ландшафте
- **Локальный min** — $f(x^*) \leq f(x)$ только в ε-окрестности — самая низкая точка в своей «ямке», но не обязательно на всей поверхности

## Функция Лагранжа

> Суть метода: мы ищем min/max функции, но нам мешают ограничения (например, «бюджет ровно 100»). Лагранж склеивает целевую функцию и ограничения в одну формулу, чтобы решать всё разом как систему уравнений.

**Общая формула** (при ограничениях $g_i(x) = 0$):

$$L(x, \lambda_0, \lambda) = \lambda_0 \cdot f(x) + \sum \lambda_i \cdot g_i(x)$$

> $f(x)$ — целевая функция (что оптимизируем). $g_i(x) = 0$ — ограничения (условия, которые обязаны выполняться). $\lambda_i$ — **множители Лагранжа** — «веса», показывающие, насколько каждое ограничение «давит» на оптимум. Чем больше |λ|, тем сильнее ограничение влияет на ответ.

**Классический частный случай** ($\lambda_0 = 1$):

$$L(x, \lambda) = f(x) + \sum \lambda_i \cdot g_i(x)$$

> В 99% задач λ₀ просто равен 1 и его не пишут. Общая формула с λ₀ нужна для теоретической полноты (случай, когда градиенты ограничений линейно зависимы). На практике всегда используют классическую формулу.

### Алгоритм решения условного экстремума:
1. Составить $L(x, \lambda)$, записать $\frac{\partial L}{\partial x_i} = 0$ и $g_i(x^*) = 0$ — решить систему → найти $x^*$
2. Проверить достаточные условия: $d^2L > 0$ → min, $d^2L < 0$ → max
3. Вычислить $f(x^*)$

### Пример

**Дано:** $f(x) = x_1^2 + x_2^2 \to \min$, ограничение $g(x) = x_1 + x_2 - 2 = 0$

**1.** Составляем: $L = x_1^2 + x_2^2 + \lambda(x_1 + x_2 - 2)$

$$\frac{\partial L}{\partial x_1} = 2x_1 + \lambda = 0 \quad \Rightarrow \quad x_1 = -\lambda/2$$
$$\frac{\partial L}{\partial x_2} = 2x_2 + \lambda = 0 \quad \Rightarrow \quad x_2 = -\lambda/2$$
$$g: x_1 + x_2 = 2 \quad \Rightarrow \quad -\lambda = 2 \quad \Rightarrow \quad \lambda = -2$$

Отсюда: $x^* = (1, 1)$

**2.** $d^2L = 2\,dx_1^2 + 2\,dx_2^2 > 0$ → точка минимума ✓

**3.** $f(1, 1) = 1 + 1 = 2$
