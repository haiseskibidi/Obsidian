# Шаблонные теги (templatetags/)

**Суть в двух словах:** Умные ножницы и кисточки для HTML-шаблонов.

**Что делает:** 
Позволяет писать свои мини-программки на Python, которые можно вызывать прямо внутри HTML-кода через фигурные скобки `{{ }}` или `{% %}`. В нашем проекте они используются как переводчики-фильтры: берут техническое слово из базы данных (например, `in_progress` или `status`) и красиво пишут на экране для пользователя по-русски («В работе», «Статус»). Это помогает не писать кучу проверок `if` в самом HTML-коде.

**Чего тут НИКОГДА нет:** 
Обращений к базе данных (SQL запросов), сохранения данных или сложной бизнес-логики. Эти функции нужны исключительно для визуального оформления (форматирование текста, дат, валюты) перед тем, как отдать страницу пользователю.
