# Админка (admin.py)

**Роль в ресторане:** 👔 **Пульт управления директора** (Прямой доступ к складу)

**Суть в двух словах:** Пульт управления директором.

**Что делает:** 
Позволяет быстро и красиво отобразить Инфраструктурные модели (`models.py`) в виде готовых таблиц и форм редактирования. Содержит настройки фильтрации (`list_filter`), поиска (`search_fields`) и отображения связанных таблиц прямо внутри карточки объекта (через классы `TabularInline`). Эта часть нужна исключительно для администраторов сайта.

**Чего тут НИКОГДА нет:** 
Доменных объектов, расчетов скидок, HTML-шаблонов сайта или валидации пользовательских данных из фронтенда. Это встроенный, изолированный инструмент Django, работающий только с таблицами базы данных.
