# Представления (presentation/views.py)

**Суть в двух словах:** Окно выдачи заказов. Общение с браузером.

**Что делает:** 
Отвечает только за протокол HTTP. Принимает запросы от браузера (GET, POST), извлекает из них данные (например, что ввели в формочку или какой файл прикрепили), отдает эти данные в Сервис (application/services) и возвращает готовый ответ. Если это обычная страница, возвращает HTML (собрав `context` для шаблона). Если это REST API, возвращает `JsonResponse`.

**Чего тут НИКОГДА нет:** 
Бизнес-логики, проверок сроков, сохранения в базу данных. Вьюха никогда не должна делать `Ticket.objects.create(...)` или проверять, прошло ли 2 часа с момента создания заявки. Её задача — только принять HTTP-запрос и отдать HTTP-ответ.
